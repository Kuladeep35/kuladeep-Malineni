This is a simple Hello World Django App. It consists of one route and returns a JSON object with Hello World message response.

To run this project:

1. Install Django and create a virtual environment.
2. Clone this repository into your project directory.
3. Run the command “python manage.py migrate” to apply migrations.
4. Run the command “python manage.py runserver” to start the web application.
5. Go to http://127.0.0.1:8000/ and the Hello World JSON response should be displayed in the browser.
